Visual Analytics Process

The idea of visual analytics become extremely important where we are dealing with large information spaces. With the help of Visual analytics people can create a tool which will help them to analyze big data and came up with synthesized information. Visual analytics can also be used as a technique to discover unexpected phenomena while detecting the expected. It also help us to gather a timely and understandable assessment.


To gain a better understanding and knowledge from data, human interaction needs to be coupled with automatic and visual analysis method. This combination is called Visual analytics process. Heterogeneous data sources are processed first in many application scenarios. This step include preprocess and transform of the data. Then next step is choosing between visual and automatic analysis methods.
One basic characteristics of the visual analytics process is alternating between visual and automatic methods. It gives us a continuous refinement and verification of the data which leads to reach a preliminary results. If automatic analysis is performed first, models of the original data are generated using data mining methods. After that the analyst interact with the data to evaluate and refine the model with the help of visualization. On the other hand, if visual data analysis is held first, analyst use an automated analysis to confirm the generated hypotheses. To reveal insightful information about the data user interaction with the visualization is very essential. Data analysis will be very effective if the user were able to gain sufficient information from the data on demand by analyzing the data according to it value of interest, relevance, interaction model. In summary, in the visual analytics process, knowledge can be gained from visualisation, automatic analysis, aswell as the preceding interactions between visualisations, models, and the human analysts.
